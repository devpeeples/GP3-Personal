M&K

Down: S, Down arrow key
Up: W, Up arrow key
Left: A, Left arrow key
Right: D, Right arrow key

Fire Shell Shocker: Left Mouse button
Bubble Dash: Q
Bubble Shield: E
Grapple: Right Mouse button
Interact: X
Pause: Tab

Aim Shell Shocker: Right Joystick

Bubble Dash: L3

Grapple: Left Trigger
